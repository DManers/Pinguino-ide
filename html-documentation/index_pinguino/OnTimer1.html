Name

OnTimer1 - Call a function at fixed interval

Description

OnTimer1 works on all 8- and 32-bits boards.

Call a function at a fixed interval using Timer 1.

The function is called indefinitely until Int.detach() is called.

If Timer 1 is free, OnTimer1() will return the number of the interrupt, otherwise it will return "false".

Multi-call to OnTimer1() is not allowed. Function will then return "false" and produce "Error : interrupt TIMER1 is already used !" on the debug output.

If overwrite is needed, always use Int.detach() with the interrupt number before new declaration.

Since

beta 9.05

Syntax 

 void OnTimer1(void (*callback)(), u8 timediv, u16 delay);
callback

Function called each time Timer1 trigger an interrupt. No argument is allowed in this function.

timediv

INT_MICROSEC, INT_MILLISEC or INT_SEC (for ms, us and sec.)

delay

time before interrupt is triggered (using timediv as unit)

Example 

void blink13()
{
	toggle(13); // set pin 13 high if it's low and vice-versa
}
 
void setup()
{
       pinMode(13, OUTPUT);
       OnTimer1(blink13, INT_MILLISEC, 500); // Use Timer1 to toggle pin 13 every 500 ms
}
 
void loop()
{
       // whatever you want here
}
