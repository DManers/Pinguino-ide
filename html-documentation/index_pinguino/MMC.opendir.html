Name

MMC.opendir - pf_opendir in pff.c

Description

It opens an exsisting directory and creates the directory object for subsequent calls. The directory object structure can be discarded at any time without any procedure.

Syntax

FRESULT pf_opendir (DIR *dj, const char *path) 

      /* dj = Pointer to directory object to create */
      /* path = Pointer to the directory path */

Return FRESULT : 

/* FR_OK (0) -> The function succeeded. */
/* FR_DISK_ERR (1) -> The function failed due to a hard error in the disk function, a wrong FAT structure or an internal error. */
/* FR_NOT_READY (2) -> The storage device could not be initialized due to a hard error or no medium. */
/* FR_NO_FILE (3) -> Could not find the file or path. */
/* FR_NOT_ENABLED (6) -> The volume has not been mounted. */

Example 

void loop()
{
    FRESULT res;
    DIR dir;
    u8 path[]="/subd1"; // just an example - it can be "/" for the root directory
    ...
    res = MMC.opendir(&dir, path);
    ...
}
