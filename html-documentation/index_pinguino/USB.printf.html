Name

USB.printf

Description

It allows to write formatted data in a string towards a PC. In fact it calls BULK.printf( ).

Syntax

void USB.printf(const u8 *fmt, ...)

Example 

void loop()
{
u16 myresult;
// put here your calculation for myresult
    USB.printf("value= %d \r\n", myresult);
    ...
}
