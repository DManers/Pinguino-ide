Name

KS_DHT.read - Read data drom DHTxx sensor

Description

u8 KS_DHT.read ( u8 pin , KS_DHT_Data * dh )
Read temprature and humidity from DHTxx sensor

Syntax 

 KS_DHT.read( pin , &dht );
Parameters
pin - Pinguino pin to read from
&dht - Pointer to DHT structure
Returns/Output
Returns a status code and populates the &dht structure with the following
u8 sign; // sign (1=negative)
double temp; // temperature
double hum; // humidity
u8 dht_error; //global error

Example 

KSDHT22 dht;
 
bErr=KS_DHT.read(DHTPIN,&dht);
