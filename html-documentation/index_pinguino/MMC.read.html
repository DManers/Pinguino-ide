Name

MMC.read

Description

It reads the opened file. Therefore you must before open it

Syntax

FRESULT MMC.read (void* buff, u16 btr, u16* br) 

       /* buff = Pointer to the read buffer (NULL:Forward data to the stream)*/
       /* btr = Number of bytes to read */
       /* br = Pointer to number of bytes read; return br value can be < btr */

Return FRESULT : 

/* FR_OK (0) -> The function succeeded. */
/* FR_DISK_ERR (1) -> The function failed due to a hard error in the disk function, a wrong FAT structure or an internal error. */
/* FR_NOT_OPENED (5) -> The file has not been opened. */
/* FR_NOT_ENABLED (6) -> The volume has not been mounted. */

Example 

void loop()
{
  u8 bufname[13]= "mysdfile.txt";
  FRESULT FResult;
  u8 data_buffer_32[33]; // a supplementary byte is necessary to form a C-string (NULL terminated)
  u16 br;
  ...
  FResult = MMC.open(bufname);
  ...
  FResult = MMC.read(data_buffer_32, 32, &br);
  ...
}
