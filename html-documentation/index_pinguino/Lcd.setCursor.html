Name

lcd.setCursor

Description

Set the position of the cursor on the LCD.

If you use lcd.print or lcd.printf after this function, you will write starting at the set position.

Syntax

P8
void lcd.setCursor(u8 col, u8 row);
P32
void lcd.setCursor(unsigned char col, unsigned char row);
Parameters
  • col = column number (= position along a row). It starts from 0, i.e. the first column is 0 (zero).
  • row = row number (= position down a column). It starts from 0, i.e. the first row is 0 (zero).

Example 

void setup()
{
    lcd(8,9,4,5,6,7,0,0,0,0);
    lcd.begin(2,0);
    lcd.setCursor(6,0);
    lcd.print("Hi!");
    lcd.setCursor(3,1);
    lcd.print("Pinguino");
}
