Name

Ethernet.write


Description

By writing the Pinguino replies to the browser taking account of its previous request. The transmission is done thru the Enc26j80 and myip channel (see Ethernet.init)


Syntax

void Ethernet.write(u8 *, u16)

Example 

#define BUFFER_SIZE 650
static u8 buf[BUFFER_SIZE+1];
 
void loop() {
  u16 dat_p;
  ...
    dat_p = print_webpage(buf); // buf contains a html page in length = dat_p, built by print_webpage(...)
    Ethernet.write(buf,dat_p); // send web page data
  ...
}
