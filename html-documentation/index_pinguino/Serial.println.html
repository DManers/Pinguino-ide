Name

Serial.println - send a value (character, decimal..) followed by a carriage return character (ASCII 13, or '\r') and a newline character (ASCII 10, or '\n') over the serial port

Description X.4

checked 8bit PIC X.4 rev.785

Board 26j50

Bootloader v4.8

Syntax 

 Serial.println(numeral_type , kind_type);

numeral_type: (u16) 

 literal, variable

kind_type: 

 FLOAT, DEC, HEX, BYTE, OCT, BIN


Example 

float fv;
u16 num;
void setup()
{
  Serial.begin(9600);
  num=200;
  fv =4.3;
}
 
void loop()
{ 
// Send numeral through serial port 
    Serial.printf( "---- Output ----\r\n" ); 
    //Serial.printf( "Float--- ");  Serial.println( fv , FLOAT ); 
    Serial.printf( "Dec  --- " ); Serial.println( num, DEC   );   
    Serial.printf( "HEX  --- " ); Serial.println( num, HEX   );   
    Serial.printf( "BYTE --- " ); Serial.println( num, BYTE  );  
    Serial.printf( "OCT  --- " ); Serial.println( num, OCT   );   
    Serial.printf( "BIN  --- " ); Serial.println( num, BIN   );   
    delay(2000);
    num++;
}

Notice

FLOAT type can not be compiled on 8 bit PIC.

Serial.println( "some string" ) is not supported.
