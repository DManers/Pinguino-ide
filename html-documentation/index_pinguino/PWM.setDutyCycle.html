Name

PWM.setDutyCycle - set PWM duty cycle

Description

void PWM.setDutyCycle ( u8 pin, u16 duty )
Set duty cycle with 10-bits resolution, allowing 1024 PWM steps. Can use any CCP module.

Syntax 

 PWM.setDutyCycle( pin , duty );
Parameters
pin - CCP pin - 5 for CCP1 or 6 for CCP2
duty - 10 bit duty cycle (0 - 1023)
Returns/Output
None
