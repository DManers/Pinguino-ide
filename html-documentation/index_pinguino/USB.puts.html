Name

USB.puts

Description

It writes a character string or array on the USB line towards a PC. In fact it calls BULK.puts( )

Syntax

u8 USB.puts(u8 *buffer, u8 length) buffer = either a string, either an array; length =length of string or array

Example 

void loop()
{
    USB.puts("Pinguino\r\n", 10);
    ...
}
