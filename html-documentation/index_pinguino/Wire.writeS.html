Name

Wire.writeS - Write n bytes to a buffer

Description

u8 Wire.writeS ( u8 *data, u8 quantity ) - (P8 Only)
Writes the number of bytes, indicated by quantity, to the tx buffer by repeating the Wire.write operation

Syntax 

 Wire.writeS ( data, length );


Parameters
u8 *data - pointer to the byte array to write in tx buffer
u8 length - number of bytes to write
Returns/Output;
u8 number of bytes written.

Example

Master_transmitter_I2C
