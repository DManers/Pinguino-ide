Name

Serial.available - Check the serial receive buffer to know if at least one byte is available.

Description

Since
Beta

Syntax 

Serial.avalaible();
Output
return 1 if a byte is available in the receive buffer. Else return 0.

Example 

void setup()
{
pinMode(0,OUTPUT);
digitalWrite(0,HIGH);
Serial.begin(9600);  // speed of UART is 9600 bauds
}
 
void loop()
{
if (Serial.available())
    {
    if (Serial.read()=='L') digitalWrite(0,HIGH);  // if L is received set output 0
    if (Serial.read()=='D') digitalWrite(0,LOW);   // if D is received reset output 0
    }
}
