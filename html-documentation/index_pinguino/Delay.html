Name

delay

Description

Pause for NUMBER milli-seconds.

Since
Beta

Syntax

P8
void delay(unsigned long ms)
P32
void delay(u16 ms)
Parameters
  • ms = duration of delay in milliseconds
Valid values from 1 to 4294967295 miliseconds.
Returns/Output
[None]

Bugs

There is a bug at IDE Versions previous to Beta 10, that will produce a delay of 80% of the time specified. So, if it's specified 1000 (1 second delay), result will be a delay of 0,8 seconds delay. This problem is solved in Beta 10 and later IDE versions.
