Name

SD.mkfs - f_mkfs in tff.c

Description

Create a file system on the drive. Two partition modes are available FDISK or SFD. When FDISK format is specified, a primary partition occupies entire space of the physical drive is created and then a FAT volume is created on the partition. When SFD format is specified, the FAT volume starts from the first sector of the physical drive.

Syntax

FRESULT SD.mkfs (byte partition, word allocsize) 

       /* partition = Partitioning rule 0:FDISK, 1:SFD */
       /* allocsize = Allocation unit size [bytes] */

Example 

void loop() {
  FRESULT rc; // Result code
  ...
  rc=SD.mkfs(0,16384U); // a primary partition + a FAT volume are created
  ...
}
