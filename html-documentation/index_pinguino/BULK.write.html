Name

BULK.write

Description

It writes a character string or array on the USB line towards a PC. It calls the function BULK.puts.

Syntax 

u8 BULK.write(u8 *buffer, u8 length);
buffer
Character chain to be sent through USB.
Type: Either a string, either a char array.
length
Length of string or array to transmit.
Type: unsigned byte
Output
Function returns the number of bytes succesfully sent to USB.
Type: unsigned byte

Example 

char buffer[64];
 
void loop()
{
    BULK.write("Pinguino\r\n", 10);
// or
    buffer = "Pinguino\r\n";
    BULK.write(buffer, 16); //assuming the true length is less the array size
    ...
}
