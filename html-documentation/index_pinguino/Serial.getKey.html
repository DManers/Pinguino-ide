Name

Serial.getkey

Since 9.5

Description

Wait for a key to be pressed. Return ASCII code of the key pressed.

Syntax

c = Serial.getkey();

Example 

void setup()
{
    Serial.begin(9600);
}
 
void loop()
{
    u8 c;
    Serial.printf("Please, press a key\n");
    c = Serial.getkey();
    Serial.printf("You pressed key [%c]\n",c);
}
