Name

BULK.available

Description

It checks if data have been received. If yes it returns a non-zero byte, otherwise zero.

Syntax 

byte BULK.available();
Output
A value equal to zero if no chars received, or different to zero if any character received.
Type: byte

Example 

u8 receivedbyte;
char buffer[64];
...
void loop()
{
    receivedbyte=0;
 
    if(BULK.available()) 
      receivedbyte = BULK.read(buffer);   
    ...
}
