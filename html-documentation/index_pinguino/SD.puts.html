Name

SD.puts - fputs in tff.c

Description

Write a string. When the function succeeds, it returns the number of characters written.

Syntax

int fputs ( const char* str, /* Pointer to the string to be output */ FIL* fil /* Pointer to the file object */ )

Example 

FIL Fil;     // File object; Global variable
 
void loop() {
  char mystring[] = "Hello";
  ...
  SD.puts(mystring,&Fil);
  ...
}
