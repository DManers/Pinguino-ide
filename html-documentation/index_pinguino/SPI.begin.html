Name

SPI.begin

Description 

Initiate an SPI communication instance. Not totally equivalent to SPI.init
Use the global parameters :
  this_mode; by default value SPI_MODE0 0x00 ; alterable by void SPI_setDataMode(u8 mode)
  this_clock; by default value SPI_MASTER_FOSC_64 0x02 ; alterable by void SPI_setMode(u8 mode) or void SPI_setClockDivider(u8 clock)
  this_role; by default value SPI_CLOCK_DIV64 0x02 ; alterable by void SPI_setMode(u8 mode) or void SPI_setClockDivider(u8 clock)

N.B. this_clock and this role have the same value in Master Mode 

  this_bitorder; by default value SPI_MSBFIRST 1 ; alterable by void SPI_setBitOrder(u8 bitorder)
  this_phase; by default value SPI_SMPEND 1<<7 ; only alterable by changing the global value
Since Beta X

Syntax 

  SPI.begin()

Example 

  void setup()
  {
     SPI.begin()
  }
