Name

USB.available

Description

Check the USB receive buffer to know if at least one byte is available.

Syntax 

byte USB.available();
Output
if the function return non-zero, a character is available in the USB receive buffer.

Example 

u8 receivedbyte;
char buffer[64];
 
void setup() {
  pinMode(0,OUTPUT);
  digitalWrite(0,LOW);
}
 
void loop()
{
    receivedbyte=0;
    if(USB.available())
    { 
      receivedbyte = USB.read(buffer);
      if (buffer[0]=='H') digitalWrite(0,HIGH);  // set output 0 if L is received in the USB buffer
      if (buffer[0]=='L') digitalWrite(0,LOW);   // reset output 0 if D is received
    }
    ...  
}
