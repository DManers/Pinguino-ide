Name

DS18B20.readrom

Description

Reads the ROM Code from a device. This command can only be used when there is only ONE slave on the bus. It allows the bus master to read the slave’s 64-bit ROM code without using the Search ROM procedure. If this command is used when there is more than one slave present on the bus, a data collision will occur when all the slaves attempt to respond at the same time.

Since Beta 9.5

Syntax 

void DS18B20.readrom(u8 pin, u8 *romcode)
Parameters 
  • pin = pin number where 1-wire bus is connected.
  • romcode = pointer to variable for identification code of device.
Returns/Output 
  • the ROM code of the device in the variable identified by the pointer romcode.
