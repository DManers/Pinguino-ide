Name

USB.send

Description

It writes a character string or array on the USB line towards a PC. It calls the function BULK.puts. The data transmitted can be handled by a Python or C program able of USB functions.

Syntax

u8 USB.write(u8 *buffer, u8 length) buffer = either a string, either an array; length =length of string or array to transmit.

Example 

char buffer[64];
 
void loop()
{
    USB.send("Pinguino\r\n", 10);
// or
    USB.send(buffer, 16); //assuming the true length is less the array size
    ...
}

Example 2

Sending integer value. 

int level[1];
void setup()
{
level[0] = 0;
}
void loop()
{
  USB.send((uchar *)level,2);
  level[0]++;
  delay(300);
}
